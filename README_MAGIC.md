## 🎨 Проекты

### 1. **Магические Частицы** (`magic-particles.html`)
Интерактивная веб-визуализация с частицами, которые реагируют на движение мыши.

**Возможности:**
- 🎯 Частицы следуют за курсором мыши
- 💥 Клик создаёт взрыв частиц
- 🌀 Три режима: Flow (поток), Orbit (орбита), Wave (волна)
- 🎨 Случайные цвета по нажатию `R`
- 🔗 Автоматические связи между близкими частицами
- ✨ Градиентный эффект вокруг курсора

**Управление:**
- Двигайте мышью - частицы следуют за вами
- Кликните - создайте взрыв
- Пробел - смените режим
- `R` - случайные цвета
- Кнопки внизу для управления

**Как запустить:**
Просто откройте `magic-particles.html` в браузере!

### 2. **Генератор Фракталов** (`fractal_generator.py`)
Создаёт красивые математические узоры - фракталы.

**Возможности:**
- 🌀 Множество Мандельброта
- 🌊 Множество Жюлиа (классическое и случайное)
- 🚢 Фрактал "Горящий корабль"
- 🎬 Анимация множества Жюлиа
- 🎨 Различные цветовые схемы

**Требования:**
```bash
pip install numpy matplotlib pillow
```

**Как запустить:**
```bash
python fractal_generator.py
```

**Примеры использования:**
- Выберите `1` для множества Мандельброта
- Выберите `2` для классического множества Жюлиа
- Выберите `3` для случайного множества Жюлиа
- Выберите `4` для фрактала "Горящий корабль"
- Выберите `5` для создания анимации
- Выберите `6` для генерации всех фракталов сразу

- **Физика в реальном времени** - частицы реагируют на ваши действия

- **Высокое качество** - генерация изображений высокого разрешения
- **Анимация** - создание GIF-анимаций фракталов

## 🚀 Дополнительные идеи для экспериментов

### Для Магических Частиц:
- Измените `particleCount` для большего/меньшего количества частиц

- Попробуйте разные цветовые карты: `'viridis'`, `'plasma'`, `'inferno'`, `'magma'`, `'hot'`
- Измените область просмотра для увеличения интересных частей
